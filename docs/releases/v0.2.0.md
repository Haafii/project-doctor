# Project Doctor v0.2.0 Release Notes

Project Doctor `0.2.0` adds live npm-backed dependency and security analysis.

## Highlights

- Added `npm audit --json` vulnerability analysis.
- Added `npm outdated --json --long` dependency drift analysis.
- Added npm registry deprecation checks for direct dependencies.
- Added confirmation-tier fixes for `npm audit fix` and `npm update`.
- Added a reusable command runner with timeouts for npm-backed checks and fixes.
- Added parser tests for npm audit, outdated-package, and deprecated-package output.

## Published Packages

```text
@haafii/project-doctor@0.2.0
@haafii/project-doctor-core@0.2.0
```

## Verify

```sh
npm view @haafii/project-doctor version
npm view @haafii/project-doctor-core version
npx @haafii/project-doctor@0.2.0 scan .
```

## Notes

The live npm checks only run when dependencies are already installed. Project Doctor will not install dependencies during a scan.
